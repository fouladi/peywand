# Changelog

This listing shows the version of the `Peywand`, the date of
release, and a summary of the changes in that release.

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.3.0] - 2026-01-03

### Added

* Add plugin for CSV import and export of database.

## [1.2.0] - 2026-01-02

### Added

* Refactoring and implement import/export functionalities for different
  file formats as "Plugins". These Plugins depend only on a callable
  session factory.

## [1.1.0] - 2026-01-01

### Added

* In addition to the `HTML-like` format, add the `JSON` format for
  exporting all entries from the database and importing data to
  the database.

## [1.0.1] - 2025-12-31

### Fixed

* Unit-Tests are fixed and updated to new arguments structure.
* README corrected.

## [1.0.0] - 2025-12-31

* Initial version.
